# Multi-Agent Stock Analysis Platform

A full-stack platform utilizing multiple AI agents to provide comprehensive stock analysis and recommendations.

## Tech Stack
- **Frontend**: React, Vite
- **Backend**: FastAPI (Python), LangGraph
- **Databases**: MongoDB (Persistent Storage), Redis (Caching)
- **Deployment**: Docker Compose

## Architecture
See [architecture.md](./architecture.md) for the detailed system design diagram.

## Project Structure
- `/backend`: FastAPI application and AI Agent orchestration.
- `/frontend`: React dashboard UI.
- `docker-compose.yml`: Orchestrates all services for easy local development.
